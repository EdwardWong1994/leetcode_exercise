##  可以使用位运算进行元素之间的交换

```
s[i]^=s[j]
s[j]^=s[i]
s[i]^=s[j]
```

常见的位运算有与/或/非/异或运算

其中`s[i]^=s[j]`表示对`s[i]`和`s[j]`进行异或运算，如果两个二进制的每一位进行比较，如果两个数相同，则结果为0，如果两个位不同，则结果为1

在`C++`中，与 -->  `&`

或 -->  `|`

非-->  `~`

在`C++`中，`&`和`&&`的区别在于：

1. `&`是按位与运算符，用于对两个整数的每一位进行逻辑与操作，并返回结果

2.  `&&`是逻辑与操作，用于对两个布尔表达式进行逻辑与操作，并返回结果
