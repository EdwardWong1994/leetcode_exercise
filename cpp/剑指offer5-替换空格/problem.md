##  为什么将下列代码稍微改动一下，反而过不了？

```
for(int i=nums1-1,j=nums-1;i>=0;i--,j--){
if(s[i]==' '){
s[j]='0';
s[j-1]='2';
s[j-2]='%';
j=j-2;
// nums--;
}else{
s[j]=s[i];
// nums--/* / */* ; */
}
}

```

逻辑不是很对，在第一层判断中，`if(s[i]=='  ')  s[nums-1]='0'` 这相当于默认最后一个是空格，但其实不一定
如果最后一个不是空格的话，会进入`s[nums--]=s[i]`的执行，但是此时应该从`s[nums-1]`填充而不是`s[nums]`填充
